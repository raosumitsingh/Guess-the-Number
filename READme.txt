A program that generate a random number and
asks the user to guess it.

If the player's guess is higher than the actual number,
the programe displays "Lower number please".
Similarly if the guess is lower than the actual number,
the program prints "Higher number please".

when the user guesses the Correct number , the program 
displays the number of attempts taken to arrive 
at the right guess.

There is also kept a RECORD of the higgest score,
which in this case is lowest number of guesses.
ONES a records breaks the user is also informed.
